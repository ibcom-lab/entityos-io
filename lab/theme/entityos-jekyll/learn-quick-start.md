## Introduction

You don't need any traditional computer science, engineering or information technology experience to use entityOS.cloud.

A key element of running any successful organisation is to be organised — and one of the simplest ways to do that is using an app built for the internet: a webapp.

This quick start guide will step you through what it takes to build a webapp using entityOS. entityOS is a fully managed service and covers all the backend infrastructure so you can focus on the organisational needs and user experience.

---

## Step 1 — How it Looks (Form)

The **form** controls how the webapp looks and is a key element of the user experience. The form is created using HTML & CSS.

[Do the Quick Start Form (HTML & CSS) tutorial →](/learn-quick-start-form)

---

## Step 2 — What it Does (Function)

The **function** controls how the webapp behaves — ie what happens when a user clicks a button. The function is constructed using JavaScript. All modern web browsers work with JavaScript code.

[Do the Quick Start Function (JavaScript) tutorial →](/learn-quick-start-function)

---

## Step 3 — Get the Tools

Install VS Code (or your preferred text editor), and install [Chrome](https://www.google.com/chrome/) or [Brave](https://www.brave.com/) for development.

---

## Step 4 — Create Your Space

Go to [console.entityos.cloud](https://console.entityos.cloud/#signup) and follow the instructions to create your own space on entityOS.

---

## Step 5 — Open the Example Webapp

Once logged in, in the **Quick Start** section click **Create Quick Start webapp**. Then open the URL for your webapp and launch Chrome Developer Tools.
